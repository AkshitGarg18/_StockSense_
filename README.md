🚀 Stock Sense

An intelligent stock analytics platform with real-time insights, AI assistance, and seamless user experience.

📌 Overview

Stock Sense is a modern stock market web application designed to provide users with real-time stock data, analytics, and AI-powered assistance.
It focuses on simplicity, performance, and smart insights to help users understand market trends effectively.

✨ Features
📊 Real-Time Stock Data
View live stock prices, changes, and market trends.
🤖 AI Chat Assistant
Get instant answers and insights related to stocks and market behavior.
🔄 Realtime Sync
Manually refresh and sync latest stock data with a single click.
📈 Analytics Dashboard
Visualize stock performance and key metrics.
📄 Export Reports (PDF/PNG)
Generate and download analytics reports instantly.
🌍 Market Status Indicator
Detect whether Indian 🇮🇳 and US 🇺🇸 markets are open or closed.
⚡ Optimized Performance
Efficient data handling using caching techniques.

🛠️ Tech Stack
Frontend: React / HTML / CSS / JavaScript
Backend: Node.js / Flask (depending on your implementation)
AI Integration: Google AI (Gemini)
APIs: Custom stock API integration
Libraries: jsPDF, html2canvas

🧠 How It Works
Fetches stock data using API
Uses caching to optimize API calls
AI assistant processes user queries
Analytics page visualizes data
Export feature generates reports dynamically
